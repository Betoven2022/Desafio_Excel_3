# 📊 Dashboard de Vendas - Xbox Game Pass Subscriptions Sales

## 📝 Descrição do Projeto

Este projeto foi desenvolvido com o objetivo de criar um **dashboard de vendas** focado na organização, análise e visualização de dados.

O principal propósito é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho das vendas e auxiliando na tomada de decisões baseadas em dados.

O dashboard apresenta indicadores e gráficos que facilitam a compreensão dos resultados de vendas dos planos do **Xbox Game Pass**, destacando métricas importantes de forma intuitiva e organizada.

---

## 🎯 Objetivos

- Organizar dados de vendas de forma estruturada;
- Transformar dados em informações visuais;
- Facilitar a análise de desempenho dos planos comercializados;
- Apoiar a tomada de decisões baseada em dados;
- Criar uma interface simples e de fácil interpretação.

---

## 📈 Funcionalidades

- Visualização do valor total de vendas;
- Filtro por tipo de plano;
- Comparação de vendas entre diferentes assinaturas;
- Indicadores de desempenho;
- Dashboard interativo e intuitivo.

## 🚀 Resultado

O dashboard proporciona uma visão consolidada das vendas de assinaturas do Xbox Game Pass, permitindo identificar rapidamente padrões, oportunidades e indicadores relevantes para o acompanhamento do negócio.
